EisenScript language

Life was like a box of chocolates. Forrest Gump

EisenScript designed by Mikael Hvidtfeldt is a design grammer for generating 3d structures using simple rule sets. This shows you even a simple system has a big possibility to generate surprising and complex structures which is generally called 'generative', and then this brings an amazing to you.

This is a javascript implementation for generating in real time in your browser.
