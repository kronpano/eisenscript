Transformations

x [float]
Move on the x axis.

y [float]
Move on the y axis.

z [float]
Move on the z axis.

rx [float]
Rotation about the x axis.

ry [float]
Rotation about the y axis.

rz [float]
Rotation about the z axis.

s [float]
Scale in the x, y and z axis by the same amount.

s [float] [float] [float]
Scale in the x, y and z axis independently.

m [f1] ... [f9]
Applies the specified 3x3 rotation matrix to the transformation matrix for the current state. About the argument order: [f1],[f2],[f3] defines the first row of the matrix.

Color space transformations

h / hue [float]
Add num to the hue value, wrapping from 360° to 0°.

sat [float]
Add num to the saturation value, wrapping from 0 to 1.

b / brightness [float]
Add num to the brightness value, wrapping from 0 to 1.

a / alpha [float]
Add num to the brightness value, wrapping from 0 to 1.

color [color]
Set specified color.

blend [color] [strength]
Blends the current color with the specified color.

set color random
Chooses a random color.
